NetChart integration example

To integrate NetChart into your web application, follow these steps:
  1. Copy "lib" folder into your web application root folder (e.g. where your index.html is)
  2. Add following code snippet in your html file
    <script src="lib/zoomcharts.js"></script>
  3. Add container element in which the chart will be added 
    <div id="demo"></div>
  4. Initialize chart (copy paste following lines to your html file): 
            <script>
            var t = new NetChart({
                container: document.getElementById("demo"),
                height: 350,
                data: {
                    preloaded: {
                        nodes: [
                            {id: "foo"},
                            {id: "bar"},
                            {id: "baz"}
                        ],
                        links: [
                            {id: "foo-bar", from: "foo", to: "bar"},
                            {id: "bar-baz", from: "bar", to: "baz"},
                            {id: "baz-foo", from: "baz", to: "foo"}
                        ]
                    }
                }
            });
        </script>
  5. Congratulations. You have now successfully integrated ZoomCharts into your web application if you can see the same chart as below in the live demo.
